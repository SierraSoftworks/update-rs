# update-rs
**A self-contained update manager for Rust applications**

This library provides a fully self-contained update manager for Rust applications. It implements
a multi-stage update strategy as described [here][app-updates] and is well suited to updating
command line applications (like [Git-Tool](https://git-tool.sierrasoftworks.com)).

[app-updates]: https://blog.sierrasoftworks.com/2019/10/15/app-updates/
